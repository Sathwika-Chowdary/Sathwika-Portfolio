# Data Source

## Dataset
IBM Telco Customer Churn sample dataset.

## Business context
This dataset is used to analyze customer churn behavior in a telecom setting.

## Target variable
- `Churn` indicates whether the customer left the service.

## Key fields commonly used
- customerID
- gender
- SeniorCitizen
- Partner
- Dependents
- tenure
- PhoneService
- InternetService
- Contract
- PaymentMethod
- MonthlyCharges
- TotalCharges
- Churn

## Core analysis focus
This project uses the dataset to study:
- overall churn rate
- churn by contract type
- churn by payment method
- churn by tenure
- churn by monthly charges
