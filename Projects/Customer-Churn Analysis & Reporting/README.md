# Customer Churn Analysis & Reporting

## Overview
This project analyzes customer churn using a public telco customer dataset and presents findings through SQL, reporting, and dashboard-ready business analysis.

## Business Goal
Identify churn patterns and answer:
- What is the overall churn rate?
- Which customer segments have the highest churn?
- How do contract type, tenure, and monthly charges affect churn?
- Which groups appear to be highest risk?

## Dataset
IBM Telco Customer Churn sample dataset.

## Planned Project Components
- SQL churn analysis queries
- KPI and segment-level reporting
- Dashboard-ready business insights
- Screenshots / outputs to be added after analysis

## Why this project matters
This project strengthens the portfolio by adding a customer analytics and retention-focused case study, complementing the sales, SQL, and ETL projects already included.
